# MeatProd-Analytics Pipeline

Система сквозной аналитики для мясоперерабатывающего цеха, объединяющая Data Engineering, аналитическую оптимизацию и машинное обучение. Проект решает две ключевые бизнес-задачи:

- прогнозирование расхода сырья и спроса на готовую продукцию;
- калькуляцию оптимальной себестоимости рецептур с учётом колебаний рыночных цен.

---

## Описание проекта

Проект моделирует типичный сценарий работы мясоперерабатывающего производства:

1. На основе рецептур, закупок сырья и продаж строится единая аналитическая база.
2. Рассчитывается фактическая себестоимость продукции в зависимости от цен на сырьё.
3. Анализируются рентабельность продуктов, чувствительность к росту цен и возможности оптимизации закупок.
4. Создаётся модель прогнозирования спроса, чтобы снизить риск перепроизводства или нехватки продукции.
5. Реализуется интерактивный дашборд для визуализации ключевых метрик.

---

## Цели проекта

- снизить издержки за счёт более точного планирования закупок;
- повысить рентабельность продукции;
- выявить рецептуры и товары, которые становятся нерентабельными при росте цен;
- прогнозировать спрос на неделю вперёд и сокращать потери от избытка или дефицита продукции;
- представить результаты в удобном для бизнеса интерфейсе.

---

## Основные этапы

### 1. Data Engineering

На этом этапе формируется и обрабатывается данные:

- таблица рецептур с процентным содержанием ингредиентов;
- таблица закупок сырья с ценами поставщиков;
- таблица продаж с датой, продуктом и объёмом в кг.

В проекте реализуется пайплайн:

- объединение данных из нескольких источников;
- обработка пропусков;
- приведение типов данных;
- расчёт текущей себестоимости 1 кг готовой продукции;
- формирование аналитической витрины для последующего анализа.

### 2. EDA и оптимизация

На этом этапе выполняется бизнес-аналитика:

- расчёт маржинальности по продуктам и по периодам;
- выявление товаров с высокой прибыльностью;
- поиск продуктов, которые стали нерентабельными из-за роста стоимости сырья;
- решение задачи оптимизации закупок: подбор комбинации поставщиков и сырья для минимизации себестоимости без нарушения рецептурных ограничений.

### 3. Машинное обучение

Создаётся модель прогнозирования спроса по временным рядам или регрессии.

В качестве базовой модели можно использовать:

- линейную регрессию;
- Random Forest Regressor;
- другие модели из scikit-learn.

Прогноз строится на основе признаков:

- день недели;
- месяц;
- тренд;
- скользящее среднее за прошлые 3 дня;
- исторические продажи по продукту.

### 4. Визуализация и деплой

Проект дополняется интерактивным дашбордом на Streamlit с вкладками:

- Аналитика цен;
- Прогноз спроса;
- Анализ рентабельности и себестоимости.

---

## Метрики качества модели

Для оценки модели используются:

- MAE (Mean Absolute Error) — средняя абсолютная ошибка в килограммах;
- RMSE (Root Mean Squared Error) — квадратный корень из средней квадратичной ошибки.

Почему эти метрики важны:

- MAE показывает, насколько в среднем ошибается прогноз по абсолютной величине;
- RMSE сильнее penalizes большие ошибки, что особенно важно для планирования производства и закупок;
- для бизнеса это означает меньшее количество лишней продукции и меньше потерь из-за недопроизводства.

## Установка

1. Клонируйте репозиторий:

```bash
git clone <repository-url>
cd MeatProd-Analytics-Pipeline
```

2. Создайте виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

Если файла requirements.txt ещё нет, его можно создать позже на основе используемых библиотек.

## Запуск проекта

### Запуск аналитики и обучения модели

```bash
python scripts/train_model.py
```

### Запуск дашборда Streamlit

```bash
streamlit run app/app.py
```

## Формат данных

Ожидается наличие следующих таблиц:

- recipes.csv — рецептуры продукции;
- supplier_prices.csv — закупки сырья и цены;
- sales.csv — история продаж;
- optional: production.csv — данные по производству.

Каждая таблица должна содержать достаточно данных для построения временного ряда и расчёта себестоимости.
